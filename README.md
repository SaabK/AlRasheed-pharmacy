# 💊 Al-Rasheed Pharmacy Website

A modern, responsive, and high-performance pharmacy website built using **HTML**, **CSS**, and **Vanilla JavaScript**. This project was created to provide Al-Rasheed Pharmacy with a professional online presence while delivering a smooth shopping experience through a clean UI, fast performance, and WhatsApp-based ordering.

The website combines elegant design, accessibility, and usability to help customers easily browse medicines and healthcare products, add items to a shopping cart, and place orders directly through WhatsApp.
